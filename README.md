# Day 02 — Navigation Bar

A clean navigation bar in dark and light themes, built with pure HTML and CSS.

## Preview
![Navbar](navbarpng)

## Features
- Dark and light theme versions
- Flexbox-based layout
- Hover effects on nav links
- Active link highlight
- CTA button

## Tech Stack
- HTML5
- CSS3 (Flexbox, transitions)

## What I Learned
- Flexbox for horizontal navigation layout
- CSS transitions for smooth hover effects
- Theming with separate CSS classes
